# MD Lens

A `cat` for **Markdown** — *styled text*, syntax-highlighted code, [hyperlinks](https://github.com/benelog/md-lens), tables, and true inline images.

## Code

```go
func greet(name string) string {
	return fmt.Sprintf("Hello, %s!", name) // syntax colors for ~15 languages
}
```

- [x] Large font headings (kitty / iTerm2 graphics)
- [x] GFM tables, task lists, blockquotes
